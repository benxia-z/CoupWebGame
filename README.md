This repo contains the source code for the Coup web game based off [the card game](https://indieboardsandcards.com/our-games/coup/).

## Getting Started

### Dependencies

- Node v18+
- npm

### Installing

#### Client:

```
cd client/CoupWebGame
npm i
```

#### Server:

```
cd server
npm i
```

### Executing program

#### Running the client locally:

```
cd client/CoupWebGame
npm run dev
```

#### Running the server locally:

```
cd server
npm run start
```
